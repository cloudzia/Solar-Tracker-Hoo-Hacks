# ☀️ SolarTrack SMA

A smart solar tracking device that uses **shape memory alloys (SMAs)** to actuate the tilting of solar panels, maximizing energy capture by following the sun throughout the day.

---

## 🔧 How It Works

Traditional solar trackers rely on motors and gears. Ours uses **shape memory alloys** — materials that bend and contract in response to heat — to passively and efficiently tilt solar panels toward the sun. This reduces mechanical complexity while maintaining responsive, real-time tracking.
